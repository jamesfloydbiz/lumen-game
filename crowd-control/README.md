# Crowd Control — FLOW ("Calm is a System")

Archived earlier prototype. A black-and-gold **crowd-safety tycoon**: steer a Marshal,
pour coins into build pads (Entry Meter / Steward Post / Widen Exit), and keep a flowing
crowd out of the red on a density heatmap. Density, not panic.

Self-contained — open `index.html` (served over http) or visit `/crowd-control/` when the
repo's root server is running. Kept for reference; the active game at the repo root is
**Monkey Breach**.
